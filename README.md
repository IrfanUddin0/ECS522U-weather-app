# ECS522U Weather App
## Installation

**Dependencies :**
```sh
npm install
```
## Setup and running:
**1. Clone this repository :**

```sh
git clone --depth 1 https://github.com/Juan-Alvarado/weatherapp-boilerplate.git weather-app
cd weather-app
```
**2. Start a live-reload development server :**
```sh
npm run dev
```
