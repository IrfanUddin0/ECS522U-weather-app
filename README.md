# ECS522U Weather App

## Dependencies :
- Node.js

## Setup and running:
**1. Clone this repository :**

```sh
git clone https://github.com/IrfanUddin0/ECS522U-weather-app
cd weather-app
```
**2. Install Dependencies**
```sh
npm install
```
**3. Start a live-reload development server :**
```sh
npm run dev
```
