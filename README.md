# ECS522U Weather App

## Dependencies :
- Node.js

## Setup and running:
**1. Clone this repository :**

```sh
git clone --depth 1 https://github.com/Juan-Alvarado/weatherapp-boilerplate.git weather-app
cd weather-app
```
**2. Install Dependencies**
```sh
npm install
```
**3. Start a live-reload development server :**
```sh
npm run dev
```
